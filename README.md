## Featured Projects

### [Intramural Sports Management App](https://github.com/uva-cs3240-s26/project-a-15)
> Full-stack web application for managing university intramural sports teams and games

- Built with **Django**, **SQLite**, and **Google OAuth** (allauth)
- Role-based access control: players, captains, and user admins
- Features: team creation, join requests, scheduling, direct messaging, free agent board
- Deployed on **Heroku** with full CI/CD pipeline via GitHub Actions
- `Python` `Django` `PostgreSQL` `HTML/CSS` `Heroku`
